# example

这是一个使用 DWeb 框架创建的项目。

## 快速开始

### 开发模式

```bash
deno task dev
```

访问 http://localhost:3000

### 构建

```bash
deno task build
```

### 生产模式

```bash
deno task start
```

## 项目结构

```
example/
├── routes/          # 路由文件（自动路由）
├── assets/          # 静态资源
├── dweb.config.ts  # 配置文件
└── deno.json       # Deno 配置
```
